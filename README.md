# TikTok CSV Content Linker (Local Version)

This contains everything you need to run your app locally.

**Privacy Note:** This is a fully local data processing tool. All CSV parsing and data aggregation happen directly in your browser. No data is uploaded to any server.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

Open http://localhost:3000 to view your dashboard.
